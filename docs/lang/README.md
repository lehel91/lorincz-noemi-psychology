# Nyelvi fájl – honlap szövegei

A `hu.json` tartalmazza a honlap összes megjelenő szövegét. Ezt a fájlt szerkesztve könnyen módosíthatók a szövegek anélkül, hogy az HTML-ben kellene bújnod.

## Szerkesztés

- Nyisd meg a **hu.json** fájlt bármilyen szövegszerkesztővel.
- A szövegek a jelenlegi tartalom szerint csoportosítva vannak (hero, rolam, miben, hogyan, rendeles, modal, stb.).
- **Új sor** a szövegben: írd `\n`-ként (pl. „Kedd\nCsütörtök”).
- **Vastagítás**: írd így: `**vastag szöveg**` (pl. „Lehetőség **elérhető**.”).
- **Telefon és e-mail** a `contact` blokkban van – ha megváltozik, itt módosítsd (phone, phoneDisplay, email).

Mentés után frissítsd az oldalt a böngészőben, és a szövegek frissülnek.
